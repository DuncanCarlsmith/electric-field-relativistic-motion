# Contributing to Electric Field Visualization

Thank you for your interest in contributing to this educational physics resource!

## How to Contribute

### Reporting Issues

If you find bugs or have suggestions for improvements:

1. Check existing issues to avoid duplicates
2. Create a new issue with:
   - Clear description of the problem or suggestion
   - Steps to reproduce (for bugs)
   - Expected vs. actual behavior
   - MATLAB version information

### Suggesting Enhancements

Enhancement suggestions are welcome! Please include:
- Clear description of the enhancement
- Educational value or use case
- Any relevant physics or pedagogical references

### Code Contributions

If you'd like to contribute code:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/YourFeature`)
3. Make your changes
4. Test thoroughly in MATLAB
5. Commit with clear messages
6. Push to your fork
7. Submit a pull request

### Guidelines

- Maintain educational clarity - code should be readable for students
- Include comments explaining physics concepts
- Test on multiple MATLAB versions if possible
- Follow MATLAB best practices for Live Scripts
- Ensure any new features have appropriate documentation

### Educational Focus

This project prioritizes:
- Pedagogical value over computational efficiency
- Clear visualization over technical complexity
- Accessibility for undergraduate physics students

## Questions?

Feel free to open an issue for any questions about contributing!
